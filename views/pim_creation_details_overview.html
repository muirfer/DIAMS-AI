AI assistant for descriptions

The AI assistant can help you to generate descriptions for your product. For more accurate results, it is recommended that the following values are completed in each relevant section before triggering the request:

  • Basic Data - Brand
  • Basic Data - Net contents of Base UoM (and Unit)
  • Basic Data - Product type
  • Purchasing Data - Purchasing group (to determine if the product is fresh or not)
  • Codes - Exclusive Lux
  • Codes - Bio
  • Codes - No gluten
  • Codes - Fair trade
  • B2C - Customer information (all languages)

The AI assistant will use the information provided in the fields above along with current task values already provided (descriptions, Base UoM, Product category, sellable units of measure, etc.) to generate the descriptions for your product.
